# Personal Portfolio Website

A modern, responsive portfolio website built to showcase my projects, skills, and experience as a Full Stack Developer.

## Live Demo
https://anasalam.vercel.app

## Tech Stack
- React
- Tailwind CSS
- Vite

## Features
- Responsive design (mobile, tablet, desktop)
- Smooth scroll-based and micro animations
- Projects showcase with live demo & GitHub links
- Sections with animated reveal
- Resume download
- Optimized for performance and accessibility

## Cloning the Repo

- Clone the repository:
```bash
git clone https://github.com/yourusername/portfolio.git
```
- Install dependencies
```bash
npm install
```
- Run development server
```bash
npm run dev
```
